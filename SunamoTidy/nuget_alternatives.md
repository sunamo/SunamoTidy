# NuGet Alternatives to SunamoTidy

This document lists popular NuGet packages that provide similar functionality to SunamoTidy.

## Overview

HTML Tidy integration

## Alternative Packages

### TidyManaged
- **NuGet**: TidyManaged
- **Purpose**: HTML Tidy wrapper
- **Key Features**: Clean and validate HTML

### HtmlAgilityPack
- **NuGet**: HtmlAgilityPack
- **Purpose**: HTML parsing and fixing
- **Key Features**: Tolerant HTML parser, DOM manipulation

### AngleSharp
- **NuGet**: AngleSharp
- **Purpose**: Standards-compliant parser
- **Key Features**: HTML5 parsing, validation

## Comparison Notes

HtmlAgilityPack for HTML parsing. AngleSharp for standards compliance.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **TidyManaged**: HTML Tidy wrapper
- **HtmlAgilityPack**: HTML parsing and fixing
- **AngleSharp**: Standards-compliant parser
